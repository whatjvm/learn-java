# 为什么要使用 maven

一个Maven项目，能够在任何IDE中快速的被编译和运行。

maven 提倡使用共同的一个目录结构标准，而且基于约定大于配置的原则，我们尽量去遵守以下目录结构:

```
源码目录:        src/main/java   
资源目录:        src/main/resources  
测试代码:        src/test/java   
测试资源:        src/test/resources  
打包输出:        target/ 
编译输出:        target/classes  
测试输出:        target/test-classes 
```
